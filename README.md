# py5_test

Un proyecto interactivo de dibujo utilizando la biblioteca `py5` (una implementación de Processing para Python). Este repositorio contiene un sketch básico que permite dibujar puntos aleatorios con colores y grosores de línea variables cuando se presiona el mouse.

## Características
- Dibuja puntos aleatorios con colores RGB generados dinámicamente.
- Ajusta el grosor de los trazos de manera aleatoria.
- Interfaz interactiva que responde al movimiento y clics del mouse.

## Requisitos
- Python 3.x
- Biblioteca `py5` instalada. Puedes instalarla con:
  ```bash
  pip install py5
  ```

## Ejecución
1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/py5_test.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd py5_test
   ```
3. Ejecuta el sketch:
   ```bash
   python test_1.py
   ```
